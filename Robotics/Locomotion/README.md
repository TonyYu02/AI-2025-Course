# Robotics / Locomotion

本文件夹用于提交 **四足/人形机器人行走** 相关作业。  
请各小组严格按照以下规范进行提交。

---

## 📂 分组目录

- 每组在本文件夹下新建自己的目录，命名为：```group-XX/```，示例：`Robotics/Locomotion/group-05/`

---

## 📌 文件结构

每个组的目录下应包含以下内容：

- **代码文件夹**：存放源代码  
- **说明文档 `README.md`**：需包含项目简要介绍、使用说明和结果展示  
- **其他必要文件**：如数据集、配置文件等  

---

## 📌 分支命名

- 每次提交作业时，请在 fork 仓库中新建分支，命名规则：```group-XX```，示例：`group-05`  
- 分支名称需与对应的组目录保持一致。

---

## 📌 提交流程

1. **Fork 仓库** 到自己的 GitHub 账号。  
2. 在 fork 仓库中新建分支（如 `group-05`）。  
3. 在本目录下新建 `group-XX/` 文件夹，放置作业文件。  
4. 提交并推送到 fork 仓库。  
5. 在 GitHub 上发起 **Pull Request**，目标分支为本仓库的 `master`。  
6. 等待审核并合并。  

---

## 📌 注意事项

- 请勿直接向 `master` 分支提交。  
- 每组仅由 **组长** 负责提交。  
- 保持目录结构和文件命名一致，否则可能被退回修改。  
- 提交前请确认文件完整且可运行。
